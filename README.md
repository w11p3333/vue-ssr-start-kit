## Intro

vue-ssr-start-kit is a simple template for building Server-Side Rendering / Progressive Web App

## Usage

### web

```bash
# install node_modules
git clone https://github.com/w11p3333/vue-ssr-start-kit.git
cd Web
npm install
# watch for dev
npm run dev
# build for production
npm run build
# run production
npm run start
```
